##  Overview

This project is a personal portfolio built for a final requirement. It features five unique sections:
- Landing Page
- About Me (Background, Education, Basic Info)
- Hobbies and Skills
- Involvement (Organizations, Community)
- Gallery of Projects

###  Key Components
- **5 Pages** with individual layouts and themes
- **Centered Hero Section** with sidebar-style navigation
- **Responsive Design** for different screen sizes
- **Font Awesome Icons** and **Google Fonts** for styling
- **Simple card design**, beginner-friendly layout
